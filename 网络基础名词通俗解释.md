## 硬件设备类

**网卡（NIC，Network Interface Card）**
就是电脑、手机里负责"插网线"或"接Wi-Fi"的那块芯片。每张网卡出厂时都有一个全球唯一的编号，叫 **MAC 地址**，相当于设备的"身份证号码"。现代电脑基本都内置了网卡，不需要另外买。

**调制解调器（Modem，俗称"猫"）**
运营商给你拉来的那根光纤或电话线，信号格式和普通网络设备不一样，"猫"负责把运营商的信号"翻译"成家里设备能读懂的网络信号。简单说：猫是"外语翻译官"，只会翻译，不会分发。

**路由器（Router）**
猫翻译完信号之后，路由器负责把这条网络连接"分享"给家里所有设备——手机、电脑、电视都通过它上网。它还给每台设备分配一个局域网内的 IP 地址（比如 192.168.1.x）。现在很多家庭用的是"光猫+路由器二合一"的设备。

**交换机（Switch）**
路由器的网线插口通常只有4个左右，如果家里有十几台电脑（比如小公司、网吧），就需要交换机来扩展网线接口。交换机只管把信号"转发"给正确的设备，不会帮你上互联网，必须配合路由器使用。类比：路由器像"门卫+小区出入口"，交换机像"楼道里的插线板"。

## 地址与标识类

**IP 地址**
设备在网络里的"门牌号"，比如 `192.168.1.5`（局域网内）或 `220.181.38.148`（公网）。局域网的 IP 是路由器分配的，临时的；公网 IP 是运营商分配给你家整个网络的，只有一个。

**MAC 地址**
设备网卡的"出厂编号"，固定不变，格式像 `A4:C3:F0:85:7A:21`。局域网内设备互相通信靠 MAC 地址，跨网络（比如访问百度）靠 IP 地址。

**网关（Gateway）**
局域网里每台设备"出门"上互联网时，都要经过一个"关口"，这个关口就是网关。在家里，网关地址通常就是路由器的 IP，比如 `192.168.1.1`。

## 网络服务类

**DNS（域名系统）**
你输入 `www.google.com`，计算机其实不认识这个名字，只认识 IP 地址。DNS 服务器就像"电话本"，帮你把域名翻译成 IP（比如 `142.250.74.68`），然后你才能连过去。

**协议（Protocol）**
网络设备之间"说话"要遵守的规则，比如：HTTP（网页传输）、TCP/IP（最底层的数据包传输规则）、Wi-Fi（无线连接标准）。就像打电话得说同一种语言，协议就是那个"语言"。

**带宽（Bandwidth）**
网络的"车道宽度"，单位是 Mbps（兆比特每秒）或 Gbps。100Mbps 意味着每秒最多传约 12MB 的数据。带宽越大，同时跑的数据越多，网速越快。

**防火墙（Firewall）**
在你的网络和外部之间设一道"门"，只让合法的数据通过，拦截可疑或危险的流量，保护内网安全。路由器里通常内置了基础防火墙功能。

---

## 一句话总结整条链路

> **互联网 → 运营商（ISP）→ 猫（Modem）→ 路由器（分配 IP）→ 交换机（扩展口）→ 你的设备（网卡接收）**

每台设备有 MAC 地址（身份证）和 IP 地址（门牌号），出门靠网关，找网站靠 DNS，传数据靠协议，安全靠防火墙。
