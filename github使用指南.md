## GitHub 入门介绍

GitHub 是全球最大的**代码托管平台**，简单来说就是一个"云端硬盘"，但专门为代码和文件设计，并加入了强大的版本管理功能。

---

### 核心概念

**Repository（仓库 / Repo）**
就像一个项目文件夹，你的所有学习资料都存放在里面。每个仓库可以设为公开（Public）或私有（Private）。

**Commit（提交）**
每次保存修改就叫一次 Commit。GitHub 会记录每一次改动，你可以随时回到任意历史版本——再也不怕误删文件了。

**Branch（分支）**
可以理解为"草稿"，在不影响主文件的情况下做实验或修改，满意了再合并进去。新手暂时不需要深入了解。

---

### 作为新手存储学习资料，你需要掌握的操作

| 操作 | 说明 |
|------|------|
| 创建仓库 | 点击右上角 `+` → New repository |
| 上传文件 | 进入仓库 → Add file → Upload files |
| 编辑文件 | 直接在网页上点击文件 → 铅笔图标编辑 |
| 查看历史 | 点击 `Commits` 查看所有修改记录 |

---

### 几个实用小建议

1. **README.md** —— 每个仓库建议创建一个 README 文件，用来写这个仓库是做什么的，方便自己日后查找。

2. **善用文件夹分类** —— 上传时可以建立层级结构，比如 `Python/笔记.md`、`数学/公式.pdf`，让资料更整洁。

3. **私有仓库** —— 个人学习资料建议设为 **Private**，只有自己能看到，免费账号也可以无限创建私有仓库。

4. **Markdown 格式** —— GitHub 原生支持 `.md` 文件渲染，用来做笔记非常美观，推荐学习基础 Markdown 语法。

## GitHub 网页界面上传资料指南

---

### 📁 创建第一个仓库

1. 登录后点击右上角 **`+`** → **New repository**
2. 填写仓库名称（建议用英文或拼音，不要有空格，用 `-` 代替）
3. 选择 **Private**（私有）
4. 勾选 **Add a README file**（自动生成说明文件）
5. 点击 **Create repository**

---

### ⬆️ 上传文件的两种方式

**方式一：直接上传**
> 进入仓库 → `Add file` → `Upload files` → 拖拽文件进去 → 滚动到底部点 `Commit changes`

**方式二：在线新建文件**
> `Add file` → `Create new file` → 输入文件名和内容 → `Commit changes`

新建文件夹的小技巧：在文件名输入框里输入 `文件夹名/文件名`，斜杠会自动创建文件夹。

---

### ⚠️ 重要注意事项

**文件大小限制**
- 单个文件最大 **25MB**（网页上传限制）
- 超过 25MB 的文件需要用 Git 命令行工具上传
- 整个仓库建议不超过 **1GB**

**不适合存储的内容**
- 视频文件（体积大，GitHub 不是视频平台）
- 大量图片压缩包
- 敏感信息（密码、身份证号等）—— 即使是私有仓库也要养成好习惯

**文件名建议**
- 避免中文文件名（部分情况下会出现乱码或路径问题）
- 不要包含空格，用 `-` 或 `_` 代替
- 例如：`python-basics.md`、`math_notes.pdf`

---

### 📝 Commit 填写建议

每次上传或修改文件，GitHub 都会要求你填写 **Commit message**（提交说明）。

| 不好的写法 | 好的写法 |
|-----------|---------|
| update | 添加 Python 第三章笔记 |
| 123 | 修正数学公式错误 |
| aaa | 上传机器学习参考资料 |

养成写清楚的习惯，日后翻历史记录会非常方便。

---

### 🗂️ 推荐的文件夹结构（参考）

```
我的学习仓库/
├── README.md          ← 仓库总说明
├── Python/
│   ├── 笔记.md
│   └── 练习代码.py
├── 数学/
│   └── 公式整理.pdf
└── 英语/
    └── 单词本.md
```

---

### 🔍 其他常用操作

- **删除文件**：点开文件 → 右上角垃圾桶图标
- **修改文件**：点开文件 → 铅笔图标（仅支持文本文件）
- **查看历史**：仓库首页点 `Commits`，可以看到每次改动记录
- **下载文件**：点开文件 → `Download raw file`
